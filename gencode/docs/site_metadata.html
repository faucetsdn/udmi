Site Metadata

Type: object
No Additional Properties

root timestamp
Type: string

RFC 3339 timestamp when the site metadata was last updated
Example:

"2019-01-17T14:02:29.364Z"

root version
Type: string

Version of the UDMI schema
Example:

"1.5.0"

root site
Type: string

Identifier for the site or building

Must match regular expression: ^[A-Z]{2}-[A-Z]{3,4}-[A-Z0-9]{2,9}$
Examples:

"UK-LON-ABC"

"US-NYC-DEF2"

root name
Type: string

Name of the site or building
Examples:

"Cheesegrater"

"Colosseum"

root tags
Type: array

Tags associated with the site

All items must be unique

Each item of this array must be:

root tags tags items
Type: object
Must match regular expression: ^[_a-z0-9]+$

Examples:

[
    "new_construction"
]

[
    "existing",
    "brownfield"
]
