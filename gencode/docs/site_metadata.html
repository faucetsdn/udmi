Site Metadata

Type: object
No Additional Properties

root timestamp
Type: string

RFC 3339 timestamp the site metadata was last updated
Example:

"2019-01-17T14:02:29.364Z"

root version
Type: string

Version of the UDMI schema

root site
Type: string

Identifier for the site or building

Must match regular expression: ^[A-Z]{2}-[A-Z]{3,4}-[A-Z0-9]{2,9}$
Examples:

"UK-LON-ABC"

"US-NYC-HUD1"

root name
Type: string

Name of the site or building
Examples:

"Cheesegrater"

"Mountain View"

Tags associated with the site
